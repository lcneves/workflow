---
title: FatalError
---

ステップ内で `FatalError` がスローされると、そのワークフローは当該ステップを再試行すべきではないことを示し、そのステップは失敗としてマークされます。

特定のステップを再試行させたくない場合にこれを使用してください。

```typescript lineNumbers
import { FatalError } from "workflow"

async function fallibleWorkflow() {
    "use workflow"
    await fallibleStep();
}

async function fallibleStep() {
    "use step"
    throw new FatalError("Fallible!") // [!code highlight]
}
```
