---
title: FatalError
---

Ketika sebuah `FatalError` dilemparkan dalam sebuah langkah, itu menunjukkan bahwa alur kerja tidak boleh mencoba ulang langkah tersebut, menandainya sebagai kegagalan.

Gunakan ini ketika Anda tidak ingin langkah tertentu dicoba ulang.

```typescript lineNumbers
import { FatalError } from "workflow"

async function fallibleWorkflow() {
    "use workflow"
    await fallibleStep();
}

async function fallibleStep() {
    "use step"
    throw new FatalError("Fallible!") // [!code highlight]
}
```
