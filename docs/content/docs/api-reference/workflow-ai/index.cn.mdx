<Callout type="warn">
该 `@workflow/ai` 包目前仍在积极开发中，应视为实验性。
</Callout>

用于将 AI SDK 集成以构建 AI 驱动的工作流的辅助工具。

## 类

<Cards>
    <Card title="DurableAgent" href="/docs/api-reference/workflow-ai/durable-agent">
      用于构建持久 AI 代理的类，能够在工作流步骤之间保持状态，并在工具执行时进行自动重试。
    </Card>
    <Card title="WorkflowChatTransport" href="/docs/api-reference/workflow-ai/workflow-chat-transport">
      为 AI SDK 提供的即插即用传输，用于在流中断时自动重连。
    </Card>
</Cards>
