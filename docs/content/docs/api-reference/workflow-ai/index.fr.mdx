Le package `@workflow/ai` est actuellement en développement actif et doit être considéré comme expérimental.

Utilitaires pour intégrer l'AI SDK afin de créer des flux de travail propulsés par l'IA.

## Classes

<Cards>
    <Card title="DurableAgent" href="/docs/api-reference/workflow-ai/durable-agent">
      Une classe pour créer des agents d'IA durables qui maintiennent l'état entre les étapes d'un flux de travail et gèrent l'exécution d'outils avec des réessais automatiques.
    </Card>
    <Card title="WorkflowChatTransport" href="/docs/api-reference/workflow-ai/workflow-chat-transport">
      Un transport plug-and-play pour l'AI SDK permettant la reconnexion automatique des flux interrompus.
    </Card>
</Cards>
