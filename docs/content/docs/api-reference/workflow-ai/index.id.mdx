<Callout type="warn">
Paket `@workflow/ai` saat ini sedang dalam pengembangan aktif dan harus dianggap eksperimental.
</Callout>

Bantuan untuk mengintegrasikan AI SDK dalam membangun alur kerja yang didukung AI.

## Kelas

<Cards>
    <Card title="DurableAgent" href="/docs/api-reference/workflow-ai/durable-agent">
      Sebuah kelas untuk membangun agen AI persisten yang mempertahankan keadaan antar langkah alur kerja dan menangani eksekusi alat dengan percobaan ulang otomatis.
    </Card>
    <Card title="WorkflowChatTransport" href="/docs/api-reference/workflow-ai/workflow-chat-transport">
      Sebuah transport drop-in untuk AI SDK yang menyediakan penyambungan ulang otomatis pada aliran yang terputus.
    </Card>
</Cards>
