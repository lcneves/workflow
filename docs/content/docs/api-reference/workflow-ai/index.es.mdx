Utilidades para integrar el SDK de IA para construir flujos de trabajo impulsados por IA.

## Clases

<Cards>
    <Card title="DurableAgent" href="/docs/api-reference/workflow-ai/durable-agent">
      Una clase para construir agentes de IA duraderos que mantienen el estado a lo largo de los pasos del flujo de trabajo y gestionan la ejecución de herramientas con reintentos automáticos.
    </Card>
    <Card title="WorkflowChatTransport" href="/docs/api-reference/workflow-ai/workflow-chat-transport">
      Un transporte listo para usar para el SDK de IA que ofrece reconexión automática en flujos interrumpidos.
    </Card>
</Cards>
