Auxiliares para integrar o SDK de IA para criar fluxos de trabalho com recursos de IA.

## Classes

<Cards>
    <Card title="DurableAgent" href="/docs/api-reference/workflow-ai/durable-agent">
      Uma classe para construir agentes de IA duráveis que mantêm estado entre etapas do fluxo de trabalho e gerenciam a execução de ferramentas com tentativas automáticas.
    </Card>
    <Card title="WorkflowChatTransport" href="/docs/api-reference/workflow-ai/workflow-chat-transport">
      Um transporte plug-and-play para o SDK de IA que proporciona reconexão automática em fluxos interrompidos.
    </Card>
</Cards>
