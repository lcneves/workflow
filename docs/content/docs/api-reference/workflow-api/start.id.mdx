Mulai/mengantri eksekusi workflow baru.

```typescript lineNumbers
import { start } from "workflow/api";
import { myWorkflow } from "./workflows/my-workflow";

const run = await start(myWorkflow); // [!code highlight]
```

## Perlu Diketahui

* Fungsi `start()` digunakan dalam konteks runtime/non-workflow untuk memicu eksekusi workflow secara programatis.
* Ini berbeda dari memanggil fungsi workflow secara langsung, yang merupakan pola tipikal dalam aplikasi Next.js.
* Fungsi ini kembali segera setelah mengantri workflow - tidak menunggu workflow selesai.
* Semua argumen harus [dapat diserialisasi](/docs/foundations/serialization).

## Contoh

### Dengan Argumen

```typescript
import { start } from "workflow/api";
import { userSignupWorkflow } from "./workflows/user-signup";

const run = await start(userSignupWorkflow, ["user@example.com"]); // [!code highlight]
```

### Dengan `StartOptions`

```typescript
import { start } from "workflow/api";
import { myWorkflow } from "./workflows/my-workflow";

const run = await start(myWorkflow, ["arg1", "arg2"], { // [!code highlight]
  deploymentId: "custom-deployment-id" // [!code highlight]
}); // [!code highlight]
```
