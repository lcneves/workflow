---
title: getRun
---

Recupera os metadados e as informações de status da execução do workflow para um determinado ID de execução. Esta função fornece acesso imediato aos detalhes da execução do workflow sem aguardar a conclusão, sendo ideal para verificação de status e monitoramento.

Use esta função quando precisar verificar o status do workflow, obter informações de tempo ou acessar metadados da execução sem bloquear até a conclusão do workflow.

```typescript lineNumbers
import { getRun } from "workflow/api";

const run = getRun("my-run-id");
```

## Exemplos

### Verificação básica de status

Verifique o status atual de uma execução do workflow:

```typescript lineNumbers
import { getRun } from "workflow/api";

export async function GET(req: Request) {
  const url = new URL(req.url);
  const runId = url.searchParams.get("runId");

  if (!runId) {
    return Response.json({ error: "No runId provided" }, { status: 400 });
  }

  try {
    const run = getRun(runId); // [!code highlight]
    const status = await run.status;

    return Response.json({ status });
  } catch (error) {
    return Response.json(
      { error: "Workflow run not found" },
      { status: 404 }
    );
  }
}
```
