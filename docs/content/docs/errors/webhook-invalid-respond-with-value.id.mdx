---
title: webhook-nilai-respondWith-tidak-valid
---

Kesalahan ini terjadi ketika Anda memberikan nilai yang tidak valid untuk opsi `respondWith` saat membuat webhook. Opsi `respondWith` harus berupa `"manual"` atau sebuah objek `Response`.

## Pesan Kesalahan

```
Invalid `respondWith` value: [value]
```

## Mengapa Ini Terjadi

Saat membuat webhook dengan `createWebhook()`, Anda dapat menentukan bagaimana webhook harus merespons permintaan HTTP yang masuk menggunakan opsi `respondWith`. Opsi ini hanya menerima nilai-nilai tertentu:

1. `"manual"` - Memungkinkan Anda mengirim respons secara manual dari dalam workflow
2. Sebuah objek `Response` - Sebuah respons yang telah ditentukan untuk dikirim segera
3. `undefined` (default) - Mengembalikan respons `202 Accepted`

## Penyebab Umum

### Menggunakan Nilai String yang Tidak Valid

```typescript lineNumbers
// Error - invalid string value
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: "automatic", // Error! // [!code highlight]
  });
}
```

**Solusi:** Gunakan `"manual"` atau berikan objek `Response`.

```typescript lineNumbers
// Fixed - use "manual"
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: "manual", // [!code highlight]
  });

  const request = await webhook;

  // Send custom response
  await request.respondWith(new Response("OK", { status: 200 })); // [!code highlight]
}
```

### Menggunakan Objek Biasa Bukan Response

```typescript lineNumbers
// Error - plain object instead of Response
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: { status: 200, body: "OK" }, // Error! // [!code highlight]
  });
}
```

**Solusi:** Buat objek `Response` yang benar.

```typescript lineNumbers
// Fixed - use Response constructor
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: new Response("OK", { status: 200 }), // [!code highlight]
  });
}
```

## Contoh Penggunaan yang Valid

### Perilaku Default (Respons 202)

```typescript lineNumbers
// Returns 202 Accepted automatically
const webhook = await createWebhook();
const request = await webhook;
// No need to send a response
```

### Respons Manual

```typescript lineNumbers
// Manual response control
const webhook = await createWebhook({
  respondWith: "manual",
});

const request = await webhook;

// Process the request...
const data = await request.json();

// Send custom response
await request.respondWith(
  new Response(JSON.stringify({ success: true }), {
    status: 200,
    headers: { "Content-Type": "application/json" },
  })
);
```

### Respons yang Telah Ditentukan

```typescript lineNumbers
// Immediate response
const webhook = await createWebhook({
  respondWith: new Response("Request received", { status: 200 }),
});

const request = await webhook;
// Response already sent
```
