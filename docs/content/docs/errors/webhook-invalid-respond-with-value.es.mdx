---
title: webhook-respondWith-valor-inválido
---

Este error ocurre cuando proporciona un valor no válido para la opción `respondWith` al crear un webhook. La opción `respondWith` debe ser o bien `"manual"` o un objeto `Response`.

## Mensaje de error

```
Invalid `respondWith` value: [value]
```

## Por qué ocurre esto

Al crear un webhook con `createWebhook()`, puede especificar cómo debe responder el webhook a las solicitudes HTTP entrantes usando la opción `respondWith`. Esta opción solo acepta valores específicos:

1. `"manual"` - Le permite enviar manualmente una respuesta desde el flujo de trabajo
2. Un objeto `Response` - Una respuesta predefinida para enviar inmediatamente
3. `undefined` (default) - Devuelve una respuesta `202 Accepted`

## Causas comunes

### Uso de un valor de cadena no válido

```typescript lineNumbers
// Error - invalid string value
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: "automatic", // Error! // [!code highlight]
  });
}
```

**Solución:** Utilice `"manual"` o proporcione un objeto `Response`.

```typescript lineNumbers
// Fixed - use "manual"
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: "manual", // [!code highlight]
  });

  const request = await webhook;

  // Send custom response
  await request.respondWith(new Response("OK", { status: 200 })); // [!code highlight]
}
```

### Uso de un objeto que no sea `Response`

```typescript lineNumbers
// Error - plain object instead of Response
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: { status: 200, body: "OK" }, // Error! // [!code highlight]
  });
}
```

**Solución:** Cree un objeto `Response` adecuado.

```typescript lineNumbers
// Fixed - use Response constructor
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: new Response("OK", { status: 200 }), // [!code highlight]
  });
}
```

## Ejemplos de uso válidos

### Comportamiento predeterminado (respuesta 202)

```typescript lineNumbers
// Returns 202 Accepted automatically
const webhook = await createWebhook();
const request = await webhook;
// No need to send a response
```

### Respuesta manual

```typescript lineNumbers
// Manual response control
const webhook = await createWebhook({
  respondWith: "manual",
});

const request = await webhook;

// Process the request...
const data = await request.json();

// Send custom response
await request.respondWith(
  new Response(JSON.stringify({ success: true }), {
    status: 200,
    headers: { "Content-Type": "application/json" },
  })
);
```

### Respuesta predefinida

```typescript lineNumbers
// Immediate response
const webhook = await createWebhook({
  respondWith: new Response("Request received", { status: 200 }),
});

const request = await webhook;
// Response already sent
```
