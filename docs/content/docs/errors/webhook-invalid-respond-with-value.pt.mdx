---
title: webhook-respondWith-valor-inválido
---

Este erro ocorre quando você fornece um valor inválido para a opção `respondWith` ao criar um webhook. A opção `respondWith` deve ser ou `"manual"` ou um objeto `Response`.

## Mensagem de Erro

```
Invalid `respondWith` value: [value]
```

## Por que isso acontece

Ao criar um webhook com `createWebhook()`, você pode especificar como o webhook deve responder às requisições HTTP recebidas usando a opção `respondWith`. Esta opção aceita apenas valores específicos:

1. `"manual"` - Permite que você envie manualmente uma resposta de dentro do fluxo de trabalho
2. Um objeto `Response` - Uma resposta pré-definida para enviar imediatamente
3. `undefined` (padrão) - Retorna uma resposta `202 Accepted`

## Causas Comuns

### Usando um valor string inválido

```typescript lineNumbers
// Error - invalid string value
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: "automatic", // Error! // [!code highlight]
  });
}
```

**Solução:** Use `"manual"` ou forneça um objeto `Response`.

```typescript lineNumbers
// Fixed - use "manual"
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: "manual", // [!code highlight]
  });

  const request = await webhook;

  // Send custom response
  await request.respondWith(new Response("OK", { status: 200 })); // [!code highlight]
}
```

### Usando um objeto que não é um Response

```typescript lineNumbers
// Error - plain object instead of Response
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: { status: 200, body: "OK" }, // Error! // [!code highlight]
  });
}
```

**Solução:** Crie um objeto `Response` apropriado.

```typescript lineNumbers
// Fixed - use Response constructor
export async function webhookWorkflow() {
  "use workflow";

  const webhook = await createWebhook({
    respondWith: new Response("OK", { status: 200 }), // [!code highlight]
  });
}
```

## Exemplos de Uso Válido

### Comportamento padrão (Resposta 202)

```typescript lineNumbers
// Returns 202 Accepted automatically
const webhook = await createWebhook();
const request = await webhook;
// No need to send a response
```

### Resposta manual

```typescript lineNumbers
// Manual response control
const webhook = await createWebhook({
  respondWith: "manual",
});

const request = await webhook;

// Process the request...
const data = await request.json();

// Send custom response
await request.respondWith(
  new Response(JSON.stringify({ success: true }), {
    status: 200,
    headers: { "Content-Type": "application/json" },
  })
);
```

### Resposta pré-definida

```typescript lineNumbers
// Immediate response
const webhook = await createWebhook({
  respondWith: new Response("Request received", { status: 200 }),
});

const request = await webhook;
// Response already sent
```
